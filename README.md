# QGIS Geospatial Analysis Project

## Overview
This project demonstrates geospatial data analysis and visualization using QGIS. The objective is to explore spatial datasets and create meaningful maps that help understand the distribution of infrastructure and services within an urban environment.

Geospatial analysis plays an important role in AI, urban planning, logistics, and location intelligence. This repository showcases GIS workflows including data preparation, spatial analysis, and map production.

---

## Project Objectives
- Import and manage geospatial datasets in QGIS
- Visualize spatial layers such as roads, buildings, and points of interest
- Perform spatial analysis including proximity and buffer analysis
- Produce clear map layouts for reporting and decision support

---

## Tools & Technologies
- **QGIS**
- **OpenStreetMap Data**

---

## Workflow
1. **Data Collection**
   - Open geospatial datasets were collected from OpenStreetMap and other public GIS sources.

2. **Data Preparation**
   - Spatial layers were cleaned and organized into meaningful categories such as roads, infrastructure, and services.

3. **Spatial Analysis**
   - Buffer analysis to determine accessibility zones around infrastructure
   - Proximity analysis to evaluate distance between facilities

4. **Visualization**
   - Layer styling and thematic mapping to highlight key spatial patterns

5. **Map Layout**
   - Final maps include legend, scale bar, north arrow, and labeled features
